# Phase 2, Plan 02: Global Hotkey & System-Wide Injection

**Objective:** Implement the global hotkey listener and "type" the transcription into the active system window.

## Requirements Addressed
- **INJS-01**: System-wide global hotkey implementation.
- **INJS-02**: Raw text injection into the active window.

## Implementation Details
- **Library:** `keyboard` for hotkeys and typing.
- **Integration:** Main app entry point (main.py) manages the hotkey state and orchestrates Audio -> STT -> Typing.
- **Config:** Store hotkey in a JSON config for late-phase user edits.
